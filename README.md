Projeto BK

Projeto front-end inspirado na interface de uma marca de fast food, com foco em layout moderno e organização visual.

Tecnologias utilizadas

- HTML5
- CSS3

Objetivo

Praticar construção de layout comercial e organização de seções de página.

O que aprendi

- Estruturação de páginas institucionais
- Organização de seções (header, banner, produtos)
- Estilização visual com CSS
- Responsividade básica
